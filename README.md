# Dotfiles

Мої налаштування Bash та Zsh.  
Цей репозиторій дозволяє швидко відновити конфігурації на новому комп’ютері.

---

## Використання

```bash
git clone https://github.com/sfrg-cell/dotfiles.git ~/dotfiles
cd ~/dotfiles

cp ~/dotfiles/.bashrc ~/
cp ~/dotfiles/.zshrc ~/

cd ~/dotfiles
git pull
